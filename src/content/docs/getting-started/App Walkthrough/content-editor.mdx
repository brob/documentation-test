The **Content editor** is where you create, edit, publish and unpublish content using the structure of the schema you previously created.

![Content editor](/images/docs/getting-started/getting-started-content.png)

The Content editor presents [default and custom views](/docs/guides/content/content-views#update-default-views) for you to access your project's contents. It also allows the possibility of reloading your content, previewing it in the API Playground, or viewing the model.

![Content editor - context menu](/images/docs/getting-started/getting-started-content-contextmenu.png)

In the Content editor you can:

- [Schedule publishing](/docs/guides/content/scheduled-publishing)
- [Create content](/docs/getting-started/create-a-content-entry)
- [Publish content](/docs/getting-started/publishing-content)
- [Update content](/docs/guides/content/updating-content)
- [Duplicate content](/docs/guides/content/duplicating-content)
- [Delete content](/docs/guides/content/deleting-content)
- [Access created content](/docs/guides/content/access-created-content)
- [Use filters](/docs/guides/content/using-filters)
- [Search for content](/docs/guides/content/searching-for-content)
- [Configure columns](/docs/guides/content/configuring-columns)
